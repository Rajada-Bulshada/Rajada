WAA MAXAY HTML?

    HTML waxaa laga soo gaabiyay Hyper Text Markup Language

    HTML waa luqada qorista ama Dhisida Web pages yada aan maalmeedka isticmaalno.

    HTML waxayna ka kooban tahay qeybo badan ama series badanoo elements ah , warbixino ah ama qoraalo ah.

    HTML elements waa midka ku amra ama usheega browser ka qaabka uu umuujin lahaa contents yada.

    HTML elements yadiisa kala gaar gaarka ah sida tusaale, "midkani waa heading", "midkani waa paragraph", "midkanna waa link", IWM.
