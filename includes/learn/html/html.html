Hordhca Html!

Hordhaca Html ka

Sidaan dhamaanteenaba uwada naqaano/lana qabsanay pages yada iyo website yada, Miyaad taqaaanaa oo miyaad ogtahay sida loo sameeyay iyo siday usoo muuqdeenba? Yuu ahaa qofka bilaabay soo saarista luqada html ka ee lawadaagida warbixinada World Wide Web ee loo yqaan (WWW)? Qeebtani iyo qeebo kale, waxaad ku baran doontaa dhamaan kala duwanaanshaheed iyo xirfada sameynta web page kaaga ku gaarka ah.

Waa maxay HTML?

html (HyperText Markup Language) waa luuqada kaliya markup loo isticmaalo sameynta ama dhisida webpages yada. Waxayna markaa diyaarisaa sida cinwaanada, madaxyada , paragraphs , liisas ,miisaska , sawir galinta, IWM. qeexida hab dhiska koowaad qoraal ahaan iyo warbixino kala duwan ee gudaha html documents ka.

  • HTML (HyperText Markup Language) waa luuqada ku Daabacaada qoraalda iyo warbixino kale duwan ee Shabakada (www).

  • HTML waa luuqada kordhinta kombiyuutarka toosan.waxaana la soo saaray 90s.

  • HTML waa aas aaska bogga. Isla markaana bogga ayaa asaas u ah Degel/Website. HTML waxay isticmaasha 'tags' si loo abuuro ama lo sameyo dukumiintyada webka.

  • HTML waa luqad HyperText markup ah; waa luqad u astaan predominant markup tag taasi oo loo adeegsado sameynta iyo nashqadeenta bogaga shabakada.

  • HTML waa luqada koowaad ee nashqadeenta Webka. CSS ayana sidoo kale waxaa loo isticmaalaa HTML si loo HTML si loo wanaajiyo nashqadeeynta iyo qurxinta boggaa Internetka. JavaScript waxaa loo isticmaalaa html si looga dhigo bogagga shabakada fir-fircoon.

  • HTML waa ay fududahay in la barto maxaa yeelay sumad kastaa waa horaa loo sii qeexay, markaa kaliya waxaan ubaahanahay inaan ogaano shaqada tags yada iyo astaamaha.

  • Bog furayaasha shabakadaha (Chrome, Internet Explorer,Firefox,Safari, iyo daalacayaasha kale ee Internet ka.) waa softiweer si loo aqriyo HTML loona muujiyo naqshadeyta bogga sida wax soo saar.

  • Waad ku qori kartaa HTML tifaf tire kasta ama editor kasta oo fudud sida Notepad, vs code, Sublim, IWM.

  • htm ama .html waa kordhin loo adeegsado in lagu qoro faylasha HTML. waxan ku qori karnaa HTML code Tifaftire/editor kasta oo ku keydin karnaa sida filename.html ama magacafeelka.html

Taariikhda Html!

Taariikhda Html ka.

HTML waxaa soo saaray/abuuray ama dhisay MudaneTim Beners-Lee dabayaaqadii 1991 laakiin si rasmi ah loo sii deyn, waxaa lagu daabacay 1995 sida HTML 2.0. HTML 4.01 waxaa la daabacay dabayaaqadii 1999 wuxuuna ahaa nooc weyn oo HTML ah.

Casharkaan wuxuu qeexayaa taariikh kooban oo HTML ah. waa luqad adeegsi aad ubaahsan oo ku lug leh nooc qumman iyo cusbooneysiin. Mudo ka hor intaan la eegin heerarkeeda iyo qeexitaankeeda, nooc kastaa wuxuu u ogolaaday adeegsiga inuu abuuro ama dhiso degel/website si fudud oo qurxoon uguna sameeyo bogagga kuwa aad wax ku ool u ah.

  • HTML 1.0 waxaa la sii daayay 1993 iyadoo looga dan lahaa in lala wadaago macluumaadka la aqrin karo oo laga heli karo daalacaashada shabakada. Balse wax badan oo kamid ah horumariyeyaashu kuma lug laheyn sameynta degellada/website yada. marka afku sidoo kale ma uusan sii kordheyn

  • Kadib wuxuu yimid HTML 2.0, oo ladaabacay 1995, kaas oo ka kooban dhamaan astaamaha HTML 1.0 oo ay wehliyaan astaamaha yar ee dheeraadka ah, kuwaas oo ku hadhay luqadda calaamadeynta caadiga ah ee loogu talagalay nashqadeynta iyo sameynta degellada ilaa janawari 1997 waxayna sifeeyeen astaamaha kala duwan ee HTML.

  • Kadib wuxuu yimid HTML 3.0, halkaasi Dave Raggettoo soo saaray waraaq cusub ama qoraal HTML ah. Waxay kamid aheyd astaamaha cusub ee HTML loo hagaajiyay, iyadoo la siinayo astaamo aad u awood badan milkiileyaasha websaydhada qaabeynta bogagga shabakada. Balse astaamahan awooddda badan ee HTML-ka cusub ayaa hoos u dhigay birawsar/browser ka iyagoo adeegsanaya horumarindheeri ah.

  • Kadib waxaa so baxay HTML 4.01, oo si balaaran loo adeegsado isla markaana ahaa nooc guuleysi oo HTML ka hor HTML 5.0, oo hada la sii daayo oo laga isticmaalo dunidoo dhan. HTML 5 waxaa loo dhihi karaa nooca dheer HTML 4.01, oo la daabacay sanadkii 2012.
